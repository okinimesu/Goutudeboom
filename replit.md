# LoL Player Ranking Application

## Overview
This is a Python Flask web application for viewing League of Legends player rankings.

## Stack
- **Language**: Python 3.11
- **Framework**: Flask
- **Dependencies**: Flask, requests

## Project Structure
```
/
├── main.py          # Main Flask application entry point
├── templates/       # HTML templates (Jinja2) - create as needed
├── static/          # Static files (CSS, JS, images) - create as needed
└── pyproject.toml   # Python project configuration
```

## Running the Application
The application runs on port 5000. Simply use the "Run" button or execute:
```bash
python main.py
```

## Configuration
- Server binds to `0.0.0.0:5000`
- Debug mode is enabled for development
